Secure Methods

POST products/adjust/quantity

Adjusts how many items remain.

POST Parameters

Parameter Name Description Example
token Your secure token "1234-5678"
SKU The SKU of the product you want to modify "abcd-efgh"
count The amount to adjust the product count by. -2
